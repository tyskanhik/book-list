
# 📚 Список книг
Приложение для управления списком книг с фильтрацией и созданием новых записей.

## ✨ Функционал
- 📖 Просмотр списка книг в виде карточек
- 🔍 Фильтрация книг:
 - Поиск по названию, описанию и автору
 - Фильтр по жанру
 - Фильтр по языку
 - Фильтр по количеству страниц (диапазон)
- ➕ Создание новых книг через форму
- 👁️ Просмотр детальной информации в модальном окне
## 🛠️ Технологии
- **Angular 17** - фреймворк
- **TypeScript** - язык программирования
- **PrimeNG** - библиотека компонентов
- **SCSS** - стилизация
- **json-server** - мок-бэкенд
- 
## 📦 Установка и запуск

1. Клонировать репозиторий:
```bash
git clone https://github.com/tyskanhik/book-list.git
cd book-list
```
2.  Установить зависимости:
```bash
npm install
```
3.  Запустить json-server (в отдельном терминале):
```bash
npm run mock
```
4.  Запустить приложение:
```bash
ng serve
```
5.  Открыть браузер:
```bash
http://localhost:4200
```
